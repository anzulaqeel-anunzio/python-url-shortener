# URL Shortener CLI

A simple command-line tool to shorten long URLs using the TinyURL API. Zero dependencies required.

<!-- Developed for Anunzio International by Anzul Aqeel. Contact +971545822608 or +971585515742 -->

## Features

*   **Fast**: Instantly generates a short link.
*   **Simple**: Just pass the URL and get the result.
*   **Zero Dependencies**: Uses Python's standard `urllib`.

## Usage

```bash
python run_shortener.py [url]
```

### Examples

**1. Basic Usage**
```bash
python run_shortener.py https://www.google.com/search?q=python+cli+tools
```
**Output:**
```
Long URL : https://www.google.com/search?q=python+cli+tools
Short URL: http://tinyurl.com/y8...
```

**2. Without Protocol**
```bash
python run_shortener.py google.com
```

## Requirements

*   Python 3.x
*   Internet connection
